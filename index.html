O que é o valor-p
Definição formal
O valor-p é a probabilidade de observar um resultado tão extremo quanto o obtido, assumindo que H₀ (hipótese nula) é verdadeira.

NÃO é a probabilidade de H₀ ser verdadeira.
NÃO é a probabilidade de ter cometido um erro.
H₀ e H₁
H₀ (nula): não há efeito, diferença ou relação.
H₁ (alternativa): há efeito, diferença ou relação.
O teste calcula: "se H₀ fosse verdadeira, qual a chance de obter este resultado?"
Como interpretar
p < 0,05 → resultado improvável sob H₀ → rejeita H₀ → resultado estatisticamente significativo
p ≥ 0,05 → resultado compatível com H₀ → não rejeita H₀

O limiar 0,05 (α) é convencional — não sagrado.
Fórmula geral
O valor-p não tem uma fórmula única — depende do teste. O que é comum é:

1. Calcular uma estatística de teste (t, F, χ²)
2. Comparar essa estatística com sua distribuição teórica
3. p = P(estatística ≥ valor observado | H₀ verdadeira)

Quanto mais extrema a estatística, menor o p.
Erros comuns
❌ p<0,05 não significa efeito grande — use tamanho de efeito (Cohen's d, η²)
❌ p>0,05 não prova que H₀ é verdadeira — é ausência de evidência, não evidência de ausência
❌ p<0,001 não é "mais significativo" — é só mais improvável sob H₀
❌ p depende do n — com n=10.000 qualquer diferença mínima será significativa
Erros tipo I e II
Erro Tipo I (α): rejeitar H₀ quando ela é verdadeira (falso positivo). Controlado por α=0,05.
Erro Tipo II (β): não rejeitar H₀ quando ela é falsa (falso negativo). Reduzido com n maior.
Poder = 1−β: probabilidade de detectar um efeito real. Típico: 0,80.

H₀: sem associação entre variáveis
χ² = Σ (O − E)² / E

Confiança—
Situação: você quer estimar a média de uma variável (ex: concentração de nitrogênio em lagos). Quanto maior a amostra, mais precisa é sua estimativa.
Pergunta: quantas unidades preciso para estimar a média com ±E de precisão?
Desvio Padrão esperado (σ)
Quão variáveis são seus dados? Obtenha da literatura ou estudo piloto. Regra: σ ≈ amplitude/4.
Maior σ → mais amostras.
Margem de erro desejada (E)
Qual precisão você aceita? E=3 significa que sua estimativa ficará dentro de ±3 unidades da média real.
Menor E → muito mais amostras.
Nível de confiança (1−α)
Em 95% das amostras possíveis, o IC conterá a média real da população.
Maior confiança → mais amostras.
Resultado e interpretação
Fórmula: n = (z × σ / E)²
z=1,96 para 95% | Arredonde sempre para cima.
Como estimar σ sem dados?
Use a literatura, estudo piloto, ou σ ≈ amplitude_esperada/4.

Poder—
Situação: você quer comparar dois grupos (ex: controle vs tratamento, antes vs depois em grupos independentes).
Pergunta: quantos indivíduos por grupo para detectar uma diferença de δ unidades com poder de 90%?
Diferença mínima relevante (δ)
Qual a menor diferença entre grupos que seria biologicamente importante? Ex: δ=5g de biomassa significa que diferenças menores não interessam.
Menor δ → muito mais amostras.
Desvio Padrão esperado (σ)
Variabilidade esperada dentro de cada grupo (use literatura ou estudo piloto).
Dados mais variáveis → mais amostras.
Poder estatístico (1−β)
Probabilidade de detectar o efeito SE ele existir. Poder baixo = risco alto de falso negativo.
Maior poder → mais amostras.
Nível de significância (α)
Risco de detectar diferença quando não há (falso positivo = erro tipo I).
Resultado e interpretação
Fórmula: n = 2σ²(z_α+z_β)²/δ²
n é por grupo. Total do estudo = 2n.
Cohen's d = δ/σ
Tamanho do efeito: <0,2 trivial | 0,2–0,5 pequeno | 0,5–0,8 médio | >0,8 grande

Erro padrão—
Situação: você quer estimar a proporção de algo (ex: % de fragmentos com espécie invasora, prevalência de uma doença, % de estudantes com determinada característica).
Pergunta: quantas unidades amostrar para estimar a proporção com ±E de precisão?
Proporção esperada (p)
Se a literatura indica ~30% de ocorrência, use p=0,30. Se não souber, use p=0,50 — é conservador e garante n suficiente para qualquer proporção.
p próximo de 0,5 → mais amostras.
Margem de erro (E)
E=0,05 significa ±5 pontos percentuais. Se a proporção real é 30%, você estimará entre 25% e 35%.
Menor E → muito mais amostras.
Nível de confiança
Resultado e interpretação
Fórmula: n = z² × p(1−p) / E²
Por que p=0,5 é conservador?
p(1−p) é máximo em p=0,5. Qualquer outra proporção dá n menor ou igual.

Poder—
Situação: você hipotiza que duas variáveis são correlacionadas (ex: temperatura × abundância de macroinvertebrados, chuva × CO₂ em rios).
Atenção: correlações fracas (r≈0,2) exigem amostras grandes. Correlações fortes (r≈0,7) precisam de menos pontos.
Correlação de Pearson esperada (r)
O r que você espera encontrar, baseado na literatura. Use valor absoluto para correlações negativas.
r=0,1 fraco | r=0,3 moderado | r=0,5 forte | r=0,7 muito forte
Menor |r| → muito mais amostras.
Nível de significância (α)
Poder estatístico (1−β)
Probabilidade de detectar a correlação se ela realmente existir.
Maior poder → mais amostras.
Resultado e interpretação
Fórmula: n = ((z_α+z_β)/Z_r)² + 3
Transformação de Fisher
Z_r = 0,5×ln((1+r)/(1−r)) — converte r para escala normal, estabilizando a variância.
Por que +3?
Correção de viés de Filon para amostras pequenas.
